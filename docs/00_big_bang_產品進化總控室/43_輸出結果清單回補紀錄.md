# 輸出結果清單回補紀錄

## 目的

輸出中心已完成三個基本軸：

- 範圍
- 內容
- 位置

本階段補上產出後體驗：輸出完成後顯示檔案清單，讓使用者可以一鍵開啟個別檔案。

## GUI 回補

### `control/gui_panels.py`

`輸出中心` 執行完成後改為開啟 `輸出中心結果` 對話框，不再只自動開資料夾。

結果清單列出：

- `資料 JSON`
- `現場統計單 Excel`
- `摘要 JSON`
- `summary PDF`
- `照片 PDF`

對話框提供：

- `開啟選取檔案`
- `開啟輸出資料夾`
- 雙擊檔案列直接開啟

若檔案路徑存在於 result 但實體檔案不存在，會以警告色標示，開啟時提示檔案不存在。
