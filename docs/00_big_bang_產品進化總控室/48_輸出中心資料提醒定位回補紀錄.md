# 輸出中心資料提醒定位回補紀錄

日期：2026-06-18

## 目的

上一階段輸出中心結果清單已能分組顯示資料提醒，但提醒仍偏「讀完自己找」。

本階段讓資料提醒成為可操作工作清單：使用者可從提醒列定位回紀錄管理頁的對應修改單。

## 調整內容

### `control/canonical_report.py`

`CanonicalReportSet.issues[]` 新增定位欄位：

- `report_id`
- `date`
- `folder`

保留原本 `report` 欄位相容舊結果。

### `control/gui_panels.py`

輸出中心結果 dialog 新增：

- `定位修改單` 按鈕
- 雙擊資料提醒列時嘗試定位修改單
- PDF 輸出列也可用 folder 嘗試定位

定位策略：

1. 優先用 `report_id`
2. 再用 `date + folder`
3. 再用 `folder`
4. 最後相容舊 `report`

如果目前紀錄清單找不到，會提示可能被篩選條件隱藏，或資料只存在 attachments/ 尚未寫入 `records.json`。

## 新增 helper

- `_focus_output_center_record(record_ref, parent=...)`
- `_find_record_tree_item_by_output_ref(record_ref)`
- `_record_matches_output_ref(record, record_ref)`
- `_output_center_issue_record_ref(issue)`
- `_format_output_center_record_ref_label(record_ref)`
- `_format_output_center_focus_missing_message(record_ref)`
