# 輸出中心結果清單分組回補紀錄

日期：2026-06-18

## 目的

輸出中心完成後，原本結果清單是平鋪檔案列表。

本階段把結果清單改成分組呈現，讓使用者能直接分辨：

- 哪些是主要交付物
- 哪些是 PDF
- 哪些是資料 JSON
- 哪些只是資料提醒

## 調整內容

### `control/gui_panels.py`

輸出中心結果 dialog 改為分組 tree：

- `主要輸出`
- `PDF`
- `資料檔`
- `資料提醒`

每筆輸出項目新增狀態語意：

- `可開啟`
- `OK，N 頁`
- `找不到檔案`
- `失敗：issue_code`
- `提醒`

若 PDF render 明確 `ok: False`，即使沒有輸出檔案路徑，也會列在結果清單中，避免失敗被靜默吞掉。

若是舊 result 沒有 `ok` 欄位且 path 為空，維持舊行為略過，避免把舊相容資料誤標成失敗。

## 新增 helper

- `_output_center_output_groups(result)`
- `_output_center_issue_items(result)`
- `_format_output_center_render_status(render, exists=...)`
- `_format_output_center_group_status(items)`

保留相容 alias：

- `_showcase_output_groups`
