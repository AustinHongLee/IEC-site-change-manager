# 輸出中心定位自動解除篩選回補紀錄

日期：2026-06-18

## 目的

上一階段資料提醒已可定位回紀錄管理頁，但如果主畫面目前被狀態篩選或搜尋框收窄，定位可能找不到目標列。

本階段讓定位流程在第一次找不到時，自動切回可見範圍：

- 狀態改回 `全部`
- 搜尋框清空
- 日期篩選欄清空
- 重新載入紀錄
- 再定位一次

這一步只改 UI 篩選狀態，不修改任何 records 或 attachments 資料。

## 調整內容

### `control/gui_panels.py`

`_focus_output_center_record(...)` 流程改為：

1. 先在目前可見紀錄清單中找目標
2. 找不到且目前篩選有收窄時，自動解除篩選並 `load_records()`
3. 再找一次
4. 仍找不到才顯示人話提示

新增 helper：

- `_reset_record_filters_for_output_focus()`
- `_output_center_filters_are_narrowed(status_text, search_text, date_from, date_to)`

提示訊息也會區分：

- 尚未解除篩選前找不到
- 已切回全部並清空搜尋後仍找不到
