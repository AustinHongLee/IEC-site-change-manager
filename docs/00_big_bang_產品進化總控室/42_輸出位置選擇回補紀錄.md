日期：2026-06-18

## 目的

輸出中心已具備：

- 輸出範圍
- 輸出內容

本階段補上第三軸：輸出位置。

## GUI 回補

### `control/gui_panels.py`

輸出中心對話框新增 `輸出位置` 區塊：

- 預設：`staging/real_attachments_showcase_gui`
- 可手動修改資料夾路徑
- 可按 `瀏覽` 選擇資料夾
- 確認訊息會即時更新輸出資料夾

執行時會把使用者指定的資料夾傳給：

```python
run_real_attachments_showcase(output_dir=...)
```

安全邊界仍沿用 showcase marker：

- 新資料夾可建立。
- 有 `.iec_real_attachments_showcase` marker 的資料夾可覆寫。
- 沒有 marker 的既有資料夾會被拒絕覆寫。
- 原始 `attachments/` 不會被修改。

## 實資料驗證

以自訂輸出位置實跑：

```powershell
python .\tools\run_real_attachments_showcase.py `
  --output .\staging\real_attachments_showcase_custom_location `
  --overwrite `
  --include 20260112/0547_AG `
  --no-summary-pdf `
  --json
```

結果：

- output: `staging/real_attachments_showcase_custom_location`
- report_count: 1
- content.statistics_xlsx: true
- content.summary_pdf: false
- content.photo_grid_pdf: true
- renders: 1 份 `photo_grid`
