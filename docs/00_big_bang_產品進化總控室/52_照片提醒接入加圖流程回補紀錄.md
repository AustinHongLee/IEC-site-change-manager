# 照片提醒接入加圖流程回補紀錄

日期：2026-06-18

## 目的

上一階段 `note` 提醒已升級為內建編輯入口。

本階段將 `before_photo` / `after_photo` 提醒接到既有加圖流程，讓使用者可直接從輸出中心結果頁補照片，而不是只開附件資料夾。

## 調整內容

### `control/gui_panels.py`

既有 `_add_image(...)` 重構為可帶建議前綴：

```python
_add_image(folder_path, preferred_prefix="")
_add_image_to_folder(folder_path, preferred_prefix="", parent=None)
```

一般從圖片明細卡片新增圖片時：

- 沒有 `preferred_prefix`
- 仍會跳出 before/after 選擇

從輸出中心資料提醒處理時：

- `before_photo` 直接使用 `before`
- `after_photo` 直接使用 `after`
- 不再要求使用者多選一次照片類型

命名規則沿用既有規則：

- 若 `before.jpg` / `after.jpg` 不存在，先寫入該檔名
- 若已存在，依序寫入 `before_1.jpg`、`before_2.jpg` 或 `after_1.jpg`、`after_2.jpg`

## issue action 變更

| issue code | 舊動作 | 新動作 |
|---|---|---|
| `before_photo` | 開啟資料夾補 before 照片 | 新增 before 照片 |
| `after_photo` | 開啟資料夾補 after 照片 | 新增 after 照片 |

## 新增 helper

- `_add_output_center_photo(record_ref, prefix, parent=...)`
- `_normalize_output_center_photo_prefix(value)`
- `_next_output_center_photo_path(folder_path, prefix)`
