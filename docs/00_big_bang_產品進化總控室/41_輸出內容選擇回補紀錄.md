# 輸出內容選擇回補紀錄

日期：2026-06-18

## 目的

上一階段已完成輸出範圍選擇；本階段補上輸出內容選擇，讓 `輸出中心` 不再固定產出所有檔案。

目前可選：

- 現場統計單 Excel
- summary PDF
- before/after 照片 PDF

## 資料層契約

### `control/real_attachments_showcase.py`

`run_real_attachments_showcase()` 新增：

```python
render_statistics=True
render_summary_pdf=True
render_photo_grid_pdf=True
```

相容舊參數：

- `render_pdf=False` 仍會關閉所有 PDF。
- 若三種輸出都關閉，會回 `ValueError`，避免產出空資料夾造成誤解。

summary 新增：

```json
{
  "content": {
    "statistics_xlsx": true,
    "summary_pdf": true,
    "photo_grid_pdf": true,
    "png": false
  }
}
```

## CLI 回補

### `tools/run_real_attachments_showcase.py`

新增：

```powershell
--no-statistics
--no-summary-pdf
--no-photo-grid-pdf
```

範例：

```powershell
python .\tools\run_real_attachments_showcase.py `
  --output .\staging\real_attachments_showcase_photo_only `
  --overwrite `
  --include 20260112/0547_AG `
  --no-statistics `
  --no-summary-pdf `
  --json
```

## GUI 回補

### `control/gui_panels.py`

紀錄管理頁原 `照片PDF` 按鈕改成 `輸出中心`。

對話框現在包含兩區：

- 輸出範圍：全部 / 目前篩選 / 選取修改單
- 輸出內容：統計單 Excel / summary PDF / before-after 照片 PDF

確認訊息會同步顯示：

- 範圍
- 預計修改單數
- 輸出內容
- 輸出資料夾

## 實資料驗證

只產照片 PDF：

```powershell
python .\tools\run_real_attachments_showcase.py `
  --output .\staging\real_attachments_showcase_photo_only `
  --overwrite `
  --include 20260112/0547_AG `
  --no-statistics `
  --no-summary-pdf `
  --json
```

結果：

- content.statistics_xlsx: false
- content.summary_pdf: false
- content.photo_grid_pdf: true
- renders: 1 份 `photo_grid`

只產統計單：

```powershell
python .\tools\run_real_attachments_showcase.py `
  --output .\staging\real_attachments_showcase_stats_only `
  --overwrite `
  --include 20260112/0547_AG `
  --no-pdf `
  --json
```

結果：

- content.statistics_xlsx: true
- content.summary_pdf: false
- content.photo_grid_pdf: false
- renders: 0
- `real_site_statistics.xlsx` 已產出
